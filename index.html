Información

Puedes utilizar número negativos para cambiar la posición de la sombra.

La mayoría de las actualizaciones resientes de los navegadores ya permiten el uso correcto de CSS3 pero es recomendable adaptar el código para que funcione en todos y no presente ningún problema, podemos utilizar “-moz-box-shadow:” para Mozilla, “webkit-box-shadow:” para Google Chrome y Safari,”box-shadow” para Explorer y Opera, PUEDES UTILIZAR TODOS JUNTOS.
